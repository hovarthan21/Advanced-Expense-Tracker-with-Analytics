##  Key Features

-  Track **Income & Expenses**
-  Category-wise expense management
-  User-based data handling using **JSON**
-  Interactive data visualization (charts)
-  Expense & income comparison analysis
-  Lightweight storage (No database required)
-  Modular and easy to extend

##  How It Works

1. User enters income or expense details
2. Data is stored securely in **JSON format**
3. Transactions are grouped by categories
4. Charts are generated to visualize spending trends
